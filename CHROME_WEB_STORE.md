- 업로드 파일: `dist/jalkiwotda-hoyoverse-v1.0.2.zip`
- Manifest: V3
- 공개 범위 권장값: `Unlisted`로 먼저 심사 제출 후, 문제 없이 통과하면 `Public` 전환

HoYoLAB 원신, 붕괴: 스타레일, 젠레스 존 제로 전적 페이지에서 캐릭터 정보를 읽어 커뮤니티 기준표 "이정도면 잘키웠다"와 비교하는 확장 프로그램입니다.

지원 기능:

- 원신, 붕괴: 스타레일, 젠레스 존 제로 전적 페이지 지원

- 여러 캐릭터를 한 번에 분석
- 공개 Google 스프레드시트 기준표와 비교
- 무기/광추/엔진, 장비 세트, 주 옵션, 주요 스탯 목표 표시
- 브라우저 안에서 HTML 리포트 생성
- 별도 서버 없이 로컬 브라우저에서 동작

사용 방법:

1. 확장 프로그램 설치 후 HoYoLAB에 로그인합니다.

## 권한 설명

### Host permission: `https://docs.google.com/*`

공개 Google 스프레드시트로 제공되는 기준표를 다운로드하기 위해 필요합니다. 확장 프로그램은 기준표 파일을 읽어 캐릭터 정보와 비교하며, 사용자의 Google 계정 데이터나 비공개 문서에 접근하지 않습니다.

### Content script matches: `https://act.hoyolab.com/*`

사용자가 직접 연 HoYoLAB 게임 전적 페이지에서 캐릭터 응답 데이터를 브라우저 안에서 읽어 리포트를 만들기 위해 필요합니다.

### Web accessible resources

HoYoLAB 페이지 컨텍스트에서 실행되는 주입 스크립트와 아이콘/이미지 자산을 불러오기 위해 필요합니다.

## 개인정보 대시보드 답변 초안

### 데이터 수집 여부

개발자 또는 개발자 서버는 사용자 데이터를 수집하지 않습니다.

확장 프로그램은 기능 제공을 위해 HoYoLAB 페이지의 캐릭터 데이터를 브라우저 로컬에서 처리하지만, 해당 데이터를 개발자에게 전송하거나 저장하지 않습니다.

### 수집 데이터 카테고리

개발자에게 전송되는 수집 데이터는 없음.

대시보드가 확장 프로그램의 로컬 처리 항목 설명을 요구하는 경우 `Website content`에 대해 "HoYoLAB 전적 페이지의 캐릭터 정보를 로컬에서 읽어 리포트 생성에만 사용하며 외부 서버로 전송하지 않음"이라고 설명합니다.

### Limited Use certification

확장 프로그램은 단일 목적 수행에 필요한 범위에서만 데이터를 사용합니다. 데이터는 판매, 광고, 신용 평가, 제3자 제공 목적으로 사용되지 않습니다.

## 심사용 메모

이 확장 프로그램은 HoYoLAB 공식 서비스가 아니며, 사용자가 직접 연 HoYoLAB 전적 페이지에서 캐릭터 정보를 읽어 공개 기준표와 비교하는 팬 메이드 도구입니다. 로그인 토큰, 비밀번호, 결제 정보는 수집하지 않습니다. 분석은 사용자의 브라우저에서 수행되며 개발자 서버로 전송되는 데이터가 없습니다.

## 제출 전 체크리스트

- `manifest.json`의 버전이 이전 제출 버전보다 높은지 확인
- `dist/jalkiwotda-hoyoverse-v1.0.2.zip` 안의 최상위에 `manifest.json`이 있는지 확인
- ZIP 안에 `.git`, `dist`, `test`, 샘플 HAR 파일이 포함되지 않았는지 확인
- 스토어 스크린샷 준비: 최소 1장, 권장 3장 이상
- 개인정보 처리방침 URL이 공개 접근 가능한지 확인
- 개발자 대시보드 Privacy practices와 `PRIVACY.md` 내용이 서로 충돌하지 않는지 확인
