# 개인정보 처리방침

최종 업데이트: 2026-06-11

`한눈에 이잘키 정오표`는 HoYoLAB 전적 페이지에서 캐릭터 정보를 읽어 공개 기준표와 비교하는 Chrome 확장 프로그램입니다.

## 수집하는 정보

개발자는 이 확장 프로그램을 통해 사용자의 개인정보, HoYoLAB 로그인 정보, 게임 UID, 캐릭터 정보, 방문 기록, 사용 통계, 분석 이벤트를 수집하거나 서버로 전송하지 않습니다.

확장 프로그램은 기능 제공을 위해 사용자의 브라우저 안에서 다음 데이터를 처리합니다.

- HoYoLAB 전적 페이지가 이미 불러온 캐릭터 응답 데이터
- 공개 Google 스프레드시트 기준표 데이터
- HoYoWiki의 공개 장비/세트 이름 데이터
- 사용자가 선택한 간단 모드, 기준표 종류, 캐릭터 정렬 정보 같은 로컬 설정

위 데이터는 리포트 생성을 위해 브라우저 로컬에서만 처리됩니다.

## 외부 통신

확장 프로그램은 다음 목적의 네트워크 요청만 수행합니다.

- `docs.google.com`: 공개 기준표 스프레드시트 다운로드
- `act.hoyolab.com` 및 HoYoLAB 관련 API: 현재 사용자가 열어 둔 전적 페이지의 캐릭터 상세 정보 확인
- HoYoWiki 공개 API: 장비/세트 이름 확인

개발자가 운영하는 별도 서버로 데이터가 전송되지 않습니다.

## 로컬 저장

확장 프로그램은 `localStorage`에 다음 로컬 설정을 저장할 수 있습니다.

- 기준표 선택값
- 간단 모드 사용 여부
- HoYoLAB 페이지에서 표시된 캐릭터 순서

이 값은 사용자의 브라우저에만 저장되며 개발자에게 전송되지 않습니다. 사용자는 브라우저의 사이트 데이터 삭제 기능 또는 확장 프로그램 삭제를 통해 해당 데이터를 제거할 수 있습니다.

## 제3자 제공

개발자는 확장 프로그램에서 처리되는 데이터를 판매, 공유, 양도하지 않습니다.
